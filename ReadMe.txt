[*ゲーム概要]
攻撃側(一人)と守備側(残り)に分かれスタートする

攻撃側は一人ずつ交代で全プレイヤーが同じ回数攻撃するように回す
攻撃側は守備側にタッチすることでタッチした人数に応じて点数が加算される
しかし画面右上のスタミナゲージが切れたとき攻撃が画面右のラインを越えていなければ攻撃側に点数は加算されない
また、守備側に捕まってしまった状態でスピードがゼロになった時も点数は加算されない
攻撃側はダッシュをしているあいだスタミナの減りが早くなる

守備側はタックルを使い攻撃側が線から右に行かないようにする
守備側のタックルが移動中の攻撃側に当たった場合、徐々に攻撃のスピードを遅くし最後は0になる
また、このとき守備側は攻撃側に引かれることになるので守備側は操作できなくなる
タックル使用後は一定時間自分の動きが止まってしまうので使い時に注意が必要だ

このゲームは実際にあるカバディというスポーツを模して作られているが、実際のカバディのルールとは違い守備側は攻撃側からの味方へのタッチがなくてもタックルで攻撃を側を捕まえることができる


[*起動方法]
ホストは端末上で 
./server ???
???の場所にプレイ人数を入れる(2-8人プレイ可)

もう一つ端末を開き
./client　clpc??? ***
???の場所にパソコンの名前(白いシールの上に記されている)
***の場所にwiiリモコンのID

ホスト以外のプレイヤーは
./client　clpc??? ***
???の場所にパソコンの名前(白いシールの上に記されている)
***の場所にwiiリモコンのID
のみでよい

wiiリモコンのIDのついては
リモコンで1,2ボタンを同時押し青色のランプが点滅を始めたら
端末上で
hcitool scan
でサーチすることができる

[*操作方法]
リモコンは横持
HOMEボタンで強制終了
攻撃側
・十字キー・・・移動
・1ボタン+十字キー・・・ダッシュ

守備側
・十字キー・・・移動
・1ボタン+十字キー・・・ダッシュ
・2ボタンタックル

[*インストール方法]
端末に
git clone https://github.com/mamorunrun/kabaddi.git
でゲームをインストールしmake
